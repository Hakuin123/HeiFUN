# HeiFUN

HeiFUN是[罗小黑妖灵论坛](www.heibbs.net)的官方应用，可在Android设备、HarmonyOS设备和Windows 11（WSA子系统）上使用。

[官方网站](https://HeiFUN.wss.cc)（待建设）

### 介绍

HeiFUN构思于2020年1月中旬，同年3月立项，5月5日开始公测。
